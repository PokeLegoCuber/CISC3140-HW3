NASA's Astronomy Picture of the Day

What is the purpose of this website?

This website builds upon my first assignment

The purpose of this website was to get Travis CI and Heroku to connect to each other. Travis CI will setup your project's enviroment and checks if there are any errors. Once if successfully builds the enviroment it talks to Heroku and deploys your application onto the web!

If it finds an error it'll halt and not deploy the appilcation.

This website demos the use of Flask(Python Web Framework) and the NASA Astronomy Picture of the Day(APOD) API

The picture on this website changes everyday

This was also my first time using the Flask Framework and an API
